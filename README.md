# 🧼 Fight Club Project

Este é um projeto de estudo focado em Front-End, utilizando a temática do filme **Clube da Luta**. O objetivo foi praticar o uso de banners com camadas (overlays) e implementação de multi-linguagem.

## 🚀 Tecnologias
- HTML5
- CSS3
